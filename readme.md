## Tech-Stack  
**Frontend:** HTML, Bootstrap v5.3  

**Backend:** Python (Flask)

**Database:** SQLite

## Features  

- Add, Complete or Delete Tasks or Projects
- Infinity Projects can be created
- Infinty Tasks can be created and linked to the related Project
- Fullscreen mode  
- Responsive template
